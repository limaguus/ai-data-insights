AI Data Insights

Projeto de análise de dados de vendas utilizando Python para gerar insights de negócio.

Objetivo
Analisar dados de vendas e identificar padrões de faturamento, produtos com melhor desempenho e categorias mais relevantes.

Tecnologias

Python
Pandas
Matplotlib
Google Colab

Análises realizadas

Faturamento total
Ticket médio
Receita por produto
Receita por categoria

Principais insights

Eletrônicos geram maior receita
Monitor é o produto mais relevante
Roupas têm potencial de aumento de ticket médio
